## Description
My own vimrc configuration based on these two repos:

- `https://github.com/mutewinter/dot_vim`
- `https://github.com/scrooloose/vimfiles`

## Installation
Clone the repo: `git clone https://github.com/guumaster/vimrc.git ~/.vim`

Grab the plugin submodules: `cd ~/.vim && git submodule init && git submodule update`

Make sure vim finds the vimrc file by either symlinking it:  `ln -s ~/.vim/vimrc ~/.vimrc`
